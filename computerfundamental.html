Class 9 Computer Science

Computer Fundamentals

Computers are electronic devices that process data to produce meaningful information. Understanding computer fundamentals is essential to learn how computers work.

1. Components of a Computer System

  • Input Devices: Devices that allow users to enter data (e.g., Keyboard, Mouse, Scanner)
  • Output Devices: Devices that display or output data (e.g., Monitor, Printer)
  • Central Processing Unit (CPU): Brain of the computer that processes data
  • Memory: Stores data temporarily (RAM) or permanently (ROM, Hard Disk)

2. Types of Computers

  • Analog Computers
  • Digital Computers
  • Hybrid Computers

3. Software

  • System Software: Helps run the computer (e.g., Operating System)
  • Application Software: Performs specific tasks (e.g., MS Word, Excel)

4. Operating System Basics

  • Manages hardware and software resources
  • Examples: Windows, Linux, macOS
  • Provides user interface for easy interaction
