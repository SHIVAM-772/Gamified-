Class 8 Computer - Cloud Computing

Unit 1: Introduction to Cloud Computing

Cloud Computing allows access to computing resources (servers, storage, databases, networking) over the internet.

Key benefits include scalability, flexibility, and cost efficiency.

Unit 2: Cloud Computing Models

There are three main service models:

  • IaaS (Infrastructure as a Service)
  • PaaS (Platform as a Service)
  • SaaS (Software as a Service)

Unit 3: Deployment Models

Cloud deployment types:

  • Public Cloud
  • Private Cloud
  • Hybrid Cloud
  • Community Cloud

Unit 4: Cloud Services

Common cloud services:

  • Storage Services (Google Drive, Dropbox)
  • Web Hosting (AWS, Azure)
  • Database Services (Firebase, Amazon RDS)
  • Collaboration Tools (MS Teams, Slack)

Unit 5: Cloud Security

Important security measures in Cloud Computing:

  • Data Encryption
  • Access Control
  • Firewall Protection
  • Regular Backups
