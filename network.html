Class 8 Computer - Networks

Unit 1: Introduction to Networks

A network is a collection of computers and devices connected together to share resources and information.

Benefits include communication, file sharing, resource sharing (printers, internet), and centralized management.

Unit 2: Types of Networks

Common network types:

  • LAN (Local Area Network)
  • WAN (Wide Area Network)
  • MAN (Metropolitan Area Network)

Unit 3: Network Topologies

Network topologies describe how devices are arranged:

  • Star
  • Bus
  • Ring

Unit 4: Network Devices

Devices used in networks:

  • Router
  • Switch
  • Hub
  • Modem

Unit 5: Network Security

Network security ensures data protection and safe communication:

  • Firewalls
  • Antivirus
  • Encryption
  • Strong passwords
